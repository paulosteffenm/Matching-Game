# Matching Game

## TL;DR

Matching game using react-native and expo-cli

## Build Setup

```bash
#Install Expo CLI
npm install --global expo-cli

#Install dependencies
npm i
```

## Run

```bash
#Web
npm run web

#Android
npm run android

#IOS
npm run ios
```
